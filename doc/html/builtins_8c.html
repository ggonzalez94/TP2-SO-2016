Detailed Description

Declaracion de funciones builtin para el programa Baash.

Contiene las funciones propias del programa. Estas son cd, help y exit.

int lsh_cd ( char **  args)

Implementacion de la funcion Change Directory.

Cambia el directorio de trabajo actual, usando la system call chdir. Retorna siempre 1 (sigue la ejecucion del Baash). Si el directorio no existe imprime un error por consola.
