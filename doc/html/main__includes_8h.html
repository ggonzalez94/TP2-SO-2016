int execute ( char **  args)

Ejecuta la instrucción ingresada.

Al presionar enter, se llama a esta funcion con los caracteres ingresados en el Baash. Si fue un string vacio, no hace nada. Caso contrario, intenta ejecutarlo como una funcion builtin. Si falla, intenta con los programas de Linux.

See also
launch()
Parameters
argsUna cadena de strings previamente parseada, lista para ser ejecutada.
Returns
0 para finalizar el Baash, 1 en caso contrario, para esperar otro comando.
