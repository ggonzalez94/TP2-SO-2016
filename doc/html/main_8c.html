Parsea el PATH del sistema Separa el contenido de la variable PATH del sistema en sus distintas rutas. Las mismas son separadas con el delimitador ":". Los tokens parseados son asignados a un arreglo de punteros a chars, variable global solo modificada en este punto, y utilizada a lo largo de todo el programa.

0 caso exitoso, -1 si no se encontro el PATH.
